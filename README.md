# ml-algorithms-from-scratch

Implementations of machine learning algorithms from scratch in Python to understand the mathematics and optimization behind them.

This repository includes both:
- Core machine learning algorithms built from scratch
- Applied projects using real-world data

## 📂 Current Work
- Linear Regression (from scratch)
- K-Means Clustering (from scratch)
- Human Activity Speed Classification (using smartphone sensor data)

## 🚀 Ongoing Work
I am continuously expanding this repository by adding more machine learning algorithms and real-world projects to strengthen my understanding of core concepts and practical applications.

## 🧠 Goals
- Build a strong foundation in machine learning fundamentals  
- Understand algorithms beyond high-level libraries  
- Apply concepts to real-world datasets
